If you want it on your own server the way to run it is:

clone the repo / download it
enter the folder
open a terminal in the folder and run pip install -r requirments.txt
go here: https://discord.com/developers/applications make an application ( bot ) and copy the bot token, app id, and bot public key
make a .env file with:

DISCORD_BOT_TOKEN=
DISCORD_APP_ID=
DISCORD_BOT_PUBLIC_KEY=

Run python3 bot.py ( maybe python command depends on your version )
Add it to your own server
